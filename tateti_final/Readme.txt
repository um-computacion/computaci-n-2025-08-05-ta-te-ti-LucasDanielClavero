Juego Ta-Te-Ti en Python
Este proyecto es una implementación del clásico juego Ta-Te-Ti (Tres en Raya) desarrollado en Python. El programa cuenta con una interfaz de consola, lógica de juego modular y pruebas unitarias para asegurar su correcto funcionamiento.

Estructura del Proyecto
Para que las instrucciones de ejecución funcionen, asegúrate de que tu proyecto mantenga la siguiente estructura de carpetas:

📁 tateti_final/
├── 📁 modelo/
│   ├── __init__.py
│   └── (archivos de la lógica del juego)
│
├── 📁 interfaz/
│   ├── __init__.py
│   └── interfaz.py
│
└── 📁 tests/
    ├── __init__.py
    └── (archivos de pruebas unitarias)

1) Abre una terminal o consola de comandos (CMD, PowerShell, Terminal, etc.).

2) Navega con el comando cd hasta la carpeta raíz del proyecto (tateti_final).

3) Una vez dentro de la carpeta tateti_final, ejecuta el siguiente comando: python -m interfaz.interfaz
